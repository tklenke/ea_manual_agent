# wikiCheck: Ignored Broken Links

Links listed here are known false positives in the broken link report — they appear in
the WR but are not real page targets. wikiCheck does not suppress these automatically;
the Reviewer reads this list and ignores them when interpreting the report.

To add a new entry: append a row to the table with the slug and the reason it appears.

| Slug | Source | Reason |
|------|--------|--------|
| `page-name` | `docs/plans/templates.md` | Placeholder slug in page template examples |
| `page-name#section-heading` | `docs/plans/templates.md` | Placeholder cross-reference in page template examples |
| `component-page-name` | `docs/plans/templates.md` | Placeholder slug in component page template examples |
| `procedure-page` | `docs/plans/templates.md` | Placeholder slug in procedure page template examples |
